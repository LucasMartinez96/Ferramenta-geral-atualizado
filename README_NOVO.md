## ✨ DESTAQUES TÉCNICOS

### Design System CSS
```css
/* Variáveis globais */
:root {
  --color-primary: #0a66c2;
  --space-md: 1rem;
  --font-family-base: -apple-system, ...;
}

/* Componentes base */
.btn, .card, .form-control, .table { }

/* Utilitários */
.p-lg, .m-md, .gap-sm, .text-center { }

/* Responsividade */
@media (max-width: 768px) { }

/* Acessibilidade */
.sr-only, :focus-visible { }
```

### Utilitários JavaScript
```javascript
// Relógio
import { HeaderClock } from './js/utils.js';
const clock = new HeaderClock('elementId');
clock.start();

// Validação
import { validators } from './js/utils.js';
if (validators.email(email)) { ... }

// Storage
import { storage } from './js/utils.js';
storage.set('key', value);

// Toast
import { Toast } from './js/utils.js';
const toast = new Toast();
toast.success('Sucesso!');
```

## 💡 DICAS PRO

### 1. Customização do Design System
Edite apenas as variáveis `:root` no CSS:
```css
:root {
  --color-primary: #SEU_COR;
  --font-family-base: 'Sua fonte';
}
```

### 2. Reutilize em outros projetos
`utils.js` é totalmente independente - copiar para qualquer projeto!

### 3. Crie componentes customizados
```javascript
// Estenda as classes disponíveis
```
